# 정적 수화 데이터 수집 번들

이 폴더는 팀원 각자의 컴퓨터에서 학습용 특징 CSV를 수집하기 위한 독립
프로그램입니다. 카메라 영상은 업로드하지 않으며 결과는 `dataset/`에 저장됩니다.

## 실행

```bash
python -m pip install -r requirements.txt
python web_server.py
```

브라우저에서 `http://localhost:8000`에 접속하고 카메라 권한을 허용합니다.
자신의 이름을 선택하면 자신에게 배정된 라벨만 작업 큐에 표시됩니다.

## 수집 규칙

- 오른손 자모와 `DELETE`는 실제 오른손으로 수집합니다.
- 겹모음 `ㅘ`, `ㅙ`, `ㅝ`, `ㅞ`는 직접 수집하지 않고 기본 모음 조합으로 만듭니다.
- 강우는 `START`(양손 펼친손)와 `END`(양손 주먹)도 수집합니다. 이 파일은 `control_features/`에 별도 저장됩니다.
- `OTHER` 클래스는 사용하지 않습니다.
- 화면은 자동 정답 판별기가 아닙니다. 선택한 라벨이 저장 CSV의 정답이 되므로 포즈를 직접 확인해야 합니다.
- `수집 시작`을 누르면 5초 준비 시간이 표시되고, 이후부터 프레임 저장이 시작됩니다.
- 목표 프레임 `180`과 저장 간격 `220ms`는 고정되어 화면에서 변경할 수 없습니다.
- 목표 프레임에 도달하면 현재 세션 저장은 자동으로 끝나고 다음 담당 라벨이 선택됩니다. 자세를 바꾼 후 `수집 시작`을 누릅니다.
- 권장량은 담당 라벨별 총 `540`프레임 이상이며, 여러 세션으로 나누어 자세와 환경을 바꿉니다.

## 담당 라벨

| 담당자 | 라벨 |
| --- | --- |
| 강우 | ㄱ, ㅈ, ㅌ, ㅐ, ㅔ, ㅜ, ㅠ, `START`, `END`, `DELETE` |
| 영래 | ㄹ, ㅅ, ㅍ, ㅑ, ㅕ, ㅡ |
| 정효 | ㄴ, ㅁ, ㅊ, ㅎ, ㅒ, ㅖ, ㅚ, ㅢ |
| 희태 | ㄷ, ㅂ, ㅇ, ㅋ, ㅏ, ㅓ, ㅗ, ㅛ, ㅟ, ㅣ |

쌍자음 `ㄲ`, `ㄸ`, `ㅃ`, `ㅆ`, `ㅉ`는 별도로 수집하지 않고 기본 자음을 두 번 입력해 만듭니다.
겹모음 `ㅘ`, `ㅙ`, `ㅝ`, `ㅞ`도 별도로 수집하지 않고 `ㅗ+ㅏ`, `ㅗ+ㅐ`,
`ㅜ+ㅓ`, `ㅜ+ㅔ`로 만듭니다. 라벨 번호는 기존 데이터 혼동을 막기 위해 유지합니다.

## 업로드

수집 완료 후 이 폴더의 `dataset/` 전체를 다음 이름으로 압축해 Google Drive
공유 폴더에 올립니다.

```text
dataset_<collector>_<YYYYMMDD>.zip
```

ZIP에는 `dataset/features/<collector>/...csv`,
`dataset/control_features/<collector>/...csv`(강우), `dataset/sessions/...json`이
포함됩니다. 영상 원본, `.DS_Store`, `__pycache__/`, `*.pyc`는 포함하지 않습니다.

이 번들은 수집 편의를 위해 collector 폴더에 저장합니다. 프로젝트 본체에 통합할 때는
`AI/dataset/features/label_XX_글자/collector__...csv` 형태의 라벨 기준 구조로 정리합니다.
